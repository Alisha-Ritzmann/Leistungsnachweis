# Projektdokumentation von Alisha Ritzmann

Das Projekt verlief gut und ich hatte nur wenige Schwierigkeiten. Ich habe den Zeitaufwand unterschätzt und brauchte länger als gedacht.

## Schwierigkeiten

In diesem Projekt hatte ich Schwierigkeiten mit den Lottie-Animationen. Ich musste die Animationen mehrmals erstellen, da ich immer wieder Fehler fand. Da ich nur fünf hochladen konnte, waren diese sehr bald aufgebraucht und ich konnte keine mehr hochladen. Leider war auch die Qualität der Animationen sehr schlecht, sodass es sehr unsauber aussah. Schlussendlich habe ich mich entschieden, diese Animationen nicht zu verwenden, sondern sie mit CSS umzusetzen.

## Änderungen zum Figma

Ich habe die Schwimmringe und das SUP in Figma so animiert, dass sie sich drehen. In meinem Projekt habe ich es so umgesetzt, dass sie so aussehen, als würden sie im Wasser schweben. Ich habe mich für diese Variante entschieden, da sie viel realistischer und ruhiger wirkt. Zusätzlich habe ich unten noch einen Button hinzugefügt, über den die User die Temperaturentwicklungen der letzten Tage einsehen können.
